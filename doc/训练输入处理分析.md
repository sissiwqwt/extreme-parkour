## 训练流程概览

### 1. 启动入口：train.py
```python
# legged_gym/scripts/train.py

def train(args):
    env, env_cfg = task_registry.make_env(name=args.task, args=args)
    ppo_runner, train_cfg = task_registry.make_alg_runner(log_root = log_pth, env=env, name=args.task, args=args)
    ppo_runner.learn(num_learning_iterations=train_cfg.runner.max_iterations, init_at_random_ep_len=True)
```

**关键步骤：**
- 创建环境（env）：生成仿真环境和depth传感器
- 创建算法runner：初始化策略网络和学习器
- 开始学习：执行learn_RL或learn_vision函数

### 2. 学习模式选择
```python
# on_policy_runner.py

class OnPolicyRunner:
    def __init__(self, ...):
        self.if_depth = self.depth_encoder_cfg["if_depth"]
        self.learn = self.learn_RL if not self.if_depth else self.learn_vision
```

- **learn_RL**：仅使用本体感觉 (Proprioception)
- **learn_vision**：使用Depth视觉信息 + 本体感觉 (推荐用于parkour任务)

## Depth图处理详解

### 1. Depth图的获取 📸

#### 1.1 环境中的相机配置
```python
# legged_gym/envs/base/legged_robot.py

self.resize_transform = torchvision.transforms.Resize(
    (self.cfg.depth.resized[1], self.cfg.depth.resized[0]),
    interpolation=torchvision.transforms.InterpolationMode.BICUBIC
)
```

**相机规格：**
- 原始分辨率：`[H, W]`（通常更大）
- 目标分辨率：58×87（压缩后）
- 使用BICUBIC插值进行缩放

#### 1.2 每个step中的Depth获取流程
```python
# step() 方法调用

def step(self, actions):
    # ... 执行物理模拟 ...
    self.post_physics_step()
    # 在post_physics_step()中调用update_depth_buffer()

def update_depth_buffer(self):
    if not self.cfg.depth.use_camera:
        return
    
    # 按照update_interval采样（不是每step都更新）
    if self.global_counter % self.cfg.depth.update_interval != 0:
        return
    
    # 1. 通过GPU相机获取原始depth数据
    self.gym.step_graphics(self.sim)
    self.gym.render_all_camera_sensors(self.sim)
    self.gym.start_access_image_tensors(self.sim)
    
    for i in range(self.num_envs):
        depth_image_ = self.gym.get_camera_image_gpu_tensor(
            self.sim, 
            self.envs[i], 
            self.cam_handles[i],
            gymapi.IMAGE_DEPTH  # 指定获取depth类型
        )
        
        depth_image = gymtorch.wrap_tensor(depth_image_)
        # 2. 图像处理
        depth_image = self.process_depth_image(depth_image, i)
        
        # 3. 更新缓冲区
        if self.episode_length_buf[i] <= 1:
            # 首帧：填充整个缓冲区
            self.depth_buffer[i] = torch.stack([depth_image] * self.cfg.depth.buffer_len, dim=0)
        else:
            # 后续帧：FIFO滑动窗口
            self.depth_buffer[i] = torch.cat([
                self.depth_buffer[i, 1:],  # 丢弃最旧的
                depth_image.to(self.device).unsqueeze(0)  # 添加最新的
            ], dim=0)
    
    self.gym.end_access_image_tensors(self.sim)
```

**关键特性：**
- ✅ **异步更新**：`update_interval`控制（如每5步更新一次）
- ✅ **缓冲区设计**：保持历史帧，形成时序信息
- ✅ **FIFO窗口**：新帧进来，旧帧淘汰

### 2. Depth图的处理 🔧

#### 2.1 图像处理流程
```python
def process_depth_image(self, depth_image, env_id):
    # 步骤1：裁剪（去除边缘噪声）
    depth_image = self.crop_depth_image(depth_image)  # [58, 79]
    
    # 步骤2：加入噪声（域随机化）
    depth_image += self.cfg.depth.dis_noise * 2 * (torch.rand(1)-0.5)[0]
    
    # 步骤3：裁剪到有效范围
    # 为什么负号？IsaacGym返回的是负的深度值
    depth_image = torch.clip(depth_image, -self.cfg.depth.far_clip, -self.cfg.depth.near_clip)
    
    # 步骤4：调整大小
    depth_image = self.resize_transform(depth_image[None, :]).squeeze()  # -> 58×87
    
    # 步骤5：归一化
    depth_image = self.normalize_depth_image(depth_image)
    
    return depth_image

def crop_depth_image(self, depth_image):
    # 左右各裁4像素，底部裁2像素（去除边界噪声）
    return depth_image[:-2, 4:-4]

def normalize_depth_image(self, depth_image):
    # 将深度值转换到[-0.5, 0.5]范围
    depth_image = depth_image * -1  # 转正值
    depth_image = (depth_image - self.cfg.depth.near_clip) / \
                  (self.cfg.depth.far_clip - self.cfg.depth.near_clip) - 0.5
    return depth_image
```

**处理管线：**
```
原始Depth → 裁剪 → 加噪声(域随机化) → 值域裁剪 → 调整分辨率 → 归一化([-0.5,0.5]) → 缓存
```

**参数含义：**
- `near_clip`：最近距离（如0.1m）
- `far_clip`：最远距离（如5.0m）
- `dis_noise`：添加的噪声幅度
- `update_interval`：更新频率

## Depth编码器处理 🧠

### 1. Depth编码器架构

#### 1.1 DepthOnlyFCBackbone58x87（CNN编码器）
```python
# rsl_rl/modules/depth_backbone.py

class DepthOnlyFCBackbone58x87(nn.Module):
    def __init__(self, prop_dim, scandots_output_dim, hidden_state_dim):
        super().__init__()
        self.image_compression = nn.Sequential(
            # 输入：[batch_size, 1, 58, 87]
            nn.Conv2d(in_channels=1, out_channels=32, kernel_size=5),
            # -> [batch_size, 32, 54, 83]
            nn.MaxPool2d(kernel_size=2, stride=2),
            # -> [batch_size, 32, 27, 41]
            nn.ELU(),
            
            nn.Conv2d(in_channels=32, out_channels=64, kernel_size=3),
            # -> [batch_size, 64, 25, 39]
            nn.ELU(),
            
            nn.Flatten(),
            # -> [batch_size, 64*25*39=61536]
            nn.Linear(64 * 25 * 39, 128),
            nn.ELU(),
            nn.Linear(128, scandots_output_dim)  # 默认32维
        )

    def forward(self, images: torch.Tensor):
        # 输入：[batch_size, 58, 87]
        images_compressed = self.image_compression(images.unsqueeze(1))
        # unsqueeze(1)添加通道维：[batch_size, 1, 58, 87]
        latent = self.output_activation(images_compressed)
        return latent  # [batch_size, 32]
```

**编码过程：**
```
[58×87] → Conv2d(5) → MaxPool(2) → Conv2d(3) → Flatten → Linear(128) → Linear(32)
```

**编码效果：**
- 原始：58×87 = 5046维
- 压缩到：32维
- **压缩比：157倍** ✨

#### 1.2 RecurrentDepthBackbone（带RNN的完整编码器）
```python
class RecurrentDepthBackbone(nn.Module):
    def __init__(self, base_backbone, env_cfg):
        super().__init__()
        self.base_backbone = base_backbone  # DepthOnlyFCBackbone58x87
        
        # 组合MLP：fusion depth features + proprioception
        self.combination_mlp = nn.Sequential(
            nn.Linear(32 + env_cfg.env.n_proprio, 128),  # 32(depth) + 53(prop) = 85
            nn.ELU(),
            nn.Linear(128, 32)
        )
        
        # RNN处理时序信息
        self.rnn = nn.GRU(
            input_size=32,
            hidden_size=512,
            batch_first=True
        )
        
        # 输出MLP
        self.output_mlp = nn.Sequential(
            nn.Linear(512, 34),  # 32(latent) + 2(yaw角)
            nn.Tanh()
        )
        self.hidden_states = None

    def forward(self, depth_image, proprioception):
        # depth_image: [batch_size, 58, 87]
        # proprioception: [batch_size, 53]
        
        # 步骤1：CNN编码深度图
        depth_latent = self.base_backbone(depth_image)
        # -> [batch_size, 32]
        
        # 步骤2：融合本体感觉信息
        depth_latent = self.combination_mlp(
            torch.cat((depth_latent, proprioception), dim=-1)
        )
        # -> [batch_size, 32]
        
        # 步骤3：RNN处理时序
        depth_latent, self.hidden_states = self.rnn(
            depth_latent[:, None, :],  # [batch, 1, 32]添加时间维
            self.hidden_states
        )
        # -> [batch, 1, 512]
        
        # 步骤4：生成输出（分离depth_latent和yaw）
        depth_latent = self.output_mlp(depth_latent.squeeze(1))
        # -> [batch, 34] = [batch, 32] + [batch, 2]
        
        return depth_latent  # [32维latent + 2维yaw角]
```

**编码流程：**
```
Depth[58×87] --CNN--> [32]
                      |
                      +--融合---> [32] --RNN--> [512] --MLP--> [34]
                      |
Proprioception[53] ---+
```

### 2. Depth在learn_vision中的使用

```python
# on_policy_runner.py - learn_vision()方法

def learn_vision(self, num_learning_iterations, init_at_random_ep_len=False):
    for it in range(self.current_learning_iteration, tot_iter):
        for i in range(self.depth_encoder_cfg["num_steps_per_env"]):
            if infos["depth"] != None:
                with torch.no_grad():
                    # 提取scan latent特征（扫描仪编码）
                    scandots_latent = self.alg.actor_critic.actor.infer_scandots_latent(obs)
                
                # 移除前向速度，保留侧向速度
                obs_prop_depth = obs[:, :self.env.cfg.env.n_proprio].clone()
                obs_prop_depth[:, 6:8] = 0  # 清空部分本体感觉信息
                
                # 编码Depth图像 + 本体感觉 -> depth_latent(32维) + yaw(2维)
                depth_latent_and_yaw = self.alg.depth_encoder(
                    infos["depth"].clone(),  # [batch, 58, 87]
                    obs_prop_depth  # [batch, 53]
                )
                
                # 分离输出
                depth_latent = depth_latent_and_yaw[:, :-2]  # [batch, 32]
                yaw = 1.5 * depth_latent_and_yaw[:, -2:]      # [batch, 2] 缩放因子
                
                depth_latent_buffer.append(depth_latent)
                yaw_buffer_student.append(yaw)
                yaw_buffer_teacher.append(obs[:, 6:8])  # 真实yaw
            
            # Student策略（使用depth信息）
            obs_student = obs.clone()
            obs_student[infos["delta_yaw_ok"], 6:8] = yaw.detach()[infos["delta_yaw_ok"]]
            
            # 调用深度独立的Actor
            actions_student = self.alg.depth_actor(
                obs_student,
                hist_encoding=True,
                scandots_latent=depth_latent  # 这里传入编码的depth特征！
            )
            
            # 环境交互
            obs, ..., = self.env.step(actions_student.detach())
        
        # 更新Depth编码器和student Actor
        depth_actor_loss, yaw_loss = self.alg.update_depth_actor(
            actions_student_buffer,
            actions_teacher_buffer,
            yaw_buffer_student,
            yaw_buffer_teacher
        )
```

**关键点：**
- ✅ Depth编码特征（32维）被当作`scandots_latent`传入Actor
- ✅ 使用Dagger算法学习：学生策略(Student)模拟教师策略(Teacher)
- ✅ 教师策略用完整观测，学生策略仅用视觉

## Actor网络的输入组成

### 完整输入拼接流程

```python
# rsl_rl/modules/actor_critic.py - Actor.forward()

def forward(self, obs, hist_encoding: bool, eval=False, scandots_latent=None):
    """
    obs结构: [prop | scan | priv_explicit | priv_latent | hist...]
    
    维度例子:
    - n_proprio = 53
    - n_scan = 被编码，维度可配置
    - n_priv_explicit = 可配置
    - n_priv_latent = 可配置
    """
    
    # 1. 分解观测向量
    obs_scan = obs[:, self.num_prop:self.num_prop + self.num_scan]
    
    # 2. 编码Scan（如LiDAR）
    if scandots_latent is None:
        scan_latent = self.scan_encoder(obs_scan)  # 编码
    else:
        scan_latent = scandots_latent  # 直接使用外部输入（来自Depth编码）
    
    # 3. 拼接Prop+Scan
    obs_prop_scan = torch.cat([obs[:, :self.num_prop], scan_latent], dim=1)
    # [53(prop) + encoded_scan_dim]
    
    # 4. 提取特权观测
    obs_priv_explicit = obs[:, 
        self.num_prop + self.num_scan:
        self.num_prop + self.num_scan + self.num_priv_explicit
    ]
    
    # 5. 编码历史 or 特权隐式信息
    if hist_encoding:
        latent = self.infer_hist_latent(obs)  # Conv1D处理历史
    else:
        latent = self.infer_priv_latent(obs)  # 特权编码器
    
    # 6. **最终拼接**
    backbone_input = torch.cat([
        obs_prop_scan,           # [53 + scan_latent_dim]
        obs_priv_explicit,       # [priv_exp_dim]
        latent                   # [priv_latent_dim 或 hist_encoded_dim]
    ], dim=1)
    
    # 7. 通过Actor主网络
    backbone_output = self.actor_backbone(backbone_input)
    return backbone_output  # [num_actions]
```

### Depth信息在Actor中的角色

| 组件 | 输入维度 | 来源 | 说明 |
|-----|---------|------|------|
| **Proprioception** | 53 | 直接观测 | 关节角度、速度、IMU |
| **Depth编码特征** | 32 | RecurrentDepthBackbone | 当`scandots_latent`注入 |
| **Yaw信息** | 2 | Depth编码器额外输出 | 估计的偏航角 |
| **权重** | 可训练 | Actor网络 | 学习如何利用视觉信息 |

## 数据流完整链路 🔄

### 训练时的完整管道

```
┌─────────────────────────────────────────────────────────────────┐
│                    环境模拟 (legged_robot.py)                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  1️⃣ GPU相机 #每update_interval步
│     ↓
│     gym.render_all_camera_sensors() → 渲染depth
│     gym.get_camera_image_gpu_tensor()
│     ↓ [原始depth值]
│
│  2️⃣ 图像处理
│     ↓ Crop: [58,79]
│     ↓ 加噪: 域随机化
│     ↓ Clip: [-far_clip, -near_clip]
│     ↓ Resize: 58×87 (BICUBIC)
│     ↓ Normalize: [-0.5, 0.5]
│     ↓ [处理后的单帧depth]
│
│  3️⃣ 缓冲区维护(FIFO)
│     depth_buffer[env_i] = [frame_t-29, ..., frame_t-1, frame_t]
│     ↓ [buffer_len, 58, 87]
│
└─────────────────────────────────────────────────────────────────┘
                           ↕️
┌─────────────────────────────────────────────────────────────────┐
│            学习器 (on_policy_runner.py - learn_vision)           │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  4️⃣ 获取Depth
│     infos["depth"] = env.depth_buffer[:, -1]
│     ↓ [batch_size, 58, 87]
│
│  5️⃣ 编码Depth (RecurrentDepthBackbone)
│     ├─ CNN编码: [58,87] → [32]
│     ├─ 融合: [32+53] → [32] (深度+本体感觉)
│     ├─ RNN: [32] → [512] (时序处理)
│     └─ 输出: [512] → [34] (32维latent + 2维yaw)
│     ↓ [batch_size, 34]
│
│  6️⃣ 提取特征
│     depth_latent = output[:, :-2]  # [batch, 32]
│     yaw = 1.5 * output[:, -2:]     # [batch, 2]
│     ↓
│
│  7️⃣ Actor前向 (Actor.forward)
│     obs_student[6:8] = yaw (用估计的yaw替代)
│     ↓
│     obs分解: prop[53] | scan | priv_explicit | priv_latent
│     ↓
│     scan_latent = depth_latent（注入的！）
│     ↓
│     拼接: [prop(53) + depth_latent(32) + priv_explicit + latent]
│     ↓
│     Actor MLP → actions
│
│  8️⃣ 更新Depth编码器
│     loss = MSE(student_actions, teacher_actions)
│           + MSE(estimated_yaw, true_yaw)
│     depth_encoder.backward()
│     ↓
│
└─────────────────────────────────────────────────────────────────┘
```

### 关键数据转换

```
原始Depth[58×87]       (5046维，图像)
    ↓
处理的Depth[58×87]     (归一化到[-0.5,0.5])
    ↓
CNN编码[32]            (157倍压缩)
    ↓

    ↓

Actor输出[动作维度]     (机器人命令)
```

## 总结：Depth处理的3个关键层次

### 💾 **第1层：获取与缓存**
- IsaacGym GPU相机每N步采集一次depth图
- 图像裁剪、去噪、归一化处理
- FIFO缓冲区维护历史帧（如30帧）

### 🧠 **第2层：编码压缩**
- **CNN**：将58×87图像压缩到32维特征
- **融合**：与本体感觉(53维)组合
- **RNN**：提取时序依赖关系
- **输出**：32维特征 + 2维yaw角度

### 🎮 **第3层：策略学习**
- Depth特征作为`scandots_latent`注入Actor
- 与proprioception/特权信息拼接
- Dagger算法优化：学生网络模拟教师网络
- 端到端学习视觉导航
