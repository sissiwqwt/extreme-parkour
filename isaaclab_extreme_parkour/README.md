# Extreme Parkour Isaac Lab Tasks

External Isaac Lab task package for the `extreme-parkour` project.

Current status:

- A1 direct-RL task scaffold
- Goal-conditioned waypoint logic
- Custom parkour terrain families:
  - `parkour_gap`
  - `beam_gap`
  - `asymmetric_gap`

Install into the Isaac Lab environment with:

```bash
cd /home/sjtu/extreme-parkour/isaaclab_extreme_parkour
pip install -e .
```
